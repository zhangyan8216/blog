## 问题分析
用户反映无法查看文章详情，经过分析发现：
1. 缺少文章详情页组件
2. 缺少文章详情页的路由配置
3. 文章卡片没有点击事件
4. API服务已经实现了获取文章详情的方法

## 解决方案

### 1. 创建文章详情页组件
- 创建 `frontend/src/pages/PostDetail.tsx` 组件
- 实现文章详情的显示逻辑
- 支持从API获取文章数据
- 保持与现有样式一致

### 2. 配置路由
- 在 `App.tsx` 中添加文章详情页路由
- 路由路径：`/posts/:filename`
- 确保路由参数正确传递

### 3. 添加文章卡片点击事件
- 在 `Home.tsx` 中为文章卡片添加点击事件
- 点击时跳转到对应的文章详情页
- 使用 `useNavigate` 进行路由跳转

### 4. 样式设计
- 确保文章详情页样式与现有设计一致
- 支持响应式布局
- 保持与文章列表卡片的视觉一致性

### 5. 测试验证
- 确保文章列表中的文章可以点击进入详情页
- 确保文章详情页数据正确显示
- 确保路由跳转正常
- 确保样式符合要求

## 技术实现要点
- 使用 React Router 的动态路由参数
- 使用 postApi.getPost 方法获取文章详情
- 保持与现有代码风格和样式的一致性
- 确保错误处理和加载状态的良好用户体验
