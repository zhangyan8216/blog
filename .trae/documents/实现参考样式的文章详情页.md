# 实现参考样式的文章详情页

## 目标
根据用户提供的参考截图，修改文章详情页的布局和样式，添加侧边栏目录、顶部横幅图片和面包屑导航等功能。

## 实现步骤

### 1. 修改 PostDetail.tsx 组件
- 添加顶部横幅图片，显示文章标题和发布信息
- 实现面包屑导航，显示当前位置
- 添加侧边栏目录组件，显示文章的章节结构
- 实现目录的高亮功能，根据当前滚动位置显示阅读进度
- 优化页面布局，采用左右分栏结构

### 2. 更新 App.scss 样式
- 添加横幅图片的样式，包括背景图片、文字排版和布局
- 实现侧边栏目录的样式，包括层级结构、高亮效果和滚动条
- 修改文章内容区域的样式，调整宽度和边距
- 优化响应式设计，确保在不同屏幕尺寸下的良好显示
- 添加目录固定定位和滚动效果

### 3. 实现目录生成逻辑
- 解析文章内容，提取标题层级结构
- 生成目录数据，包括标题文本和对应的锚点链接
- 实现滚动监听，更新当前高亮的目录项
- 添加目录项的点击事件，实现平滑滚动到对应章节

### 4. 测试和优化
- 测试长文章的目录生成和滚动高亮功能
- 验证代码块的语法高亮显示
- 测试响应式布局在不同设备上的效果
- 优化性能，确保滚动监听不会影响页面流畅度

## 技术要点
- React 组件设计和状态管理
- 滚动监听和 DOM 操作
- CSS 布局和响应式设计
- 动态目录生成和高亮
- 平滑滚动效果实现
